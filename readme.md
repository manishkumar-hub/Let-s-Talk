# Let-s-Talk
Let's Talk is multi user group chat app built using NodeJs and Socket IO .

### Built With

* [HTML, CSS , JS](https://www.w3schools.com/html/)
* [Node](https://nodejs.org/en/docs/)
* [Socket IO](https://socket.io/)
